📝 Key design decisions in this wireframe

  • Hero background: Replaced the external `themeslay.com` image with a CSS gradient — zero network cost, always available.
  • Stats row (NEW StatCard): Adds at-a-glance context (active auctions, total items, bids today) — common in auction platforms, costs minimal dev effort.
  • Filter pills (NEW FilterPills): Lets users filter by All / Active / Ending Soon / Draft without a full page reload.
  • Active nav highlight: Current page is marked in the nav (uses TanStack Router's built-in activeProps).
  • AuctionCard footer: Added countdown + Bid Now CTA directly on the card — no need to navigate away to bid.
  • Skeleton loading (NEW AuctionCardSkeleton): Replaces the jarring empty → populated flash.
  • Shared ErrorBanner (NEW): Eliminates the duplicated error banner pattern across 5 pages.
  • Focus rings: All inputs get a visible focus indicator — required for WCAG 2.4.7.
  • Responsive: Stats row 3→2→1 col, auction grid 3→2→1 col; buttons stack full-width on mobile.
